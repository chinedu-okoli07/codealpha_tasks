# CodeAlpha Network Sniffer

This project is a basic network sniffer built using Python and Scapy. It captures live network traffic and displays source and destination IP addresses along with protocol types.

##  Features
- Captures real-time network packets
- Displays source and destination IP addresses
- Identifies protocols (TCP, UDP, ICMP)

##  Tools Used
- Python
- Scapy
- Kali Linux

##  How to Run
1. Open terminal in Kali Linux
2. Navigate to the file location
3. Run:
   sudo python3 sniffer.py

##  Sample Output
[TCP] 192.168.1.5 -> 142.250.190.78  
[UDP] 192.168.1.5 -> 224.0.0.251  
[ICMP] 192.168.1.5 -> 8.8.8.8
